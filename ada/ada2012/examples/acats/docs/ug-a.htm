Annex A
Version Description

ACATS 4.2 includes 4188 tests in 4963 files, not including foundation and other support units. From Version 4.1 to 4.2, 178 tests were added, comprising 273 files (including ten foundation files). 207 tests (218 files) were modified. Three tests were split, removing three files and adding 26 files. 21 tests (30 files) were removed. One support file was added and six support files were modifed (none were deleted). One documentation file was added, 197 documentation files were modified, and one documentation file was deleted.
