4.1 Structure

The ACATS 4.2 test software includes test code that exercises specific Ada features, foundation code (used by multiple tests), support code (used to generate test results), and tool code (used to build tools necessary to customize ACATS tests). The suite includes tests for the core language and tests for the Specialized Needs Annexes. The following table summarizes the number of tests and files in the ACATS suite.
