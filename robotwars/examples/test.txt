
; ROBOT "TEST"
SCAN
   AIM TO RADAR
   IF RADAR < 0 GOSUB FIRE
   RADAR TO R
   IF R < 20 . 20 TO R
   1000 / R + AIM TO AIM
   GOTO SCAN
FOUND2
   J + AIM TO AIM
FOUND
   I + AIM TO AIM
FIRE
   0 - RADAR TO SHOT
   600 / RADAR TO I
   GOTO SEARCH
AGAIN
   0 - RADAR TO SHOT

SEARCH
   0 - I TO I
   AIM + I TO RADAR
   IF RADAR < 0 GOTO FOUND
   0 - I TO I
   AIM + I TO RADAR
   IF RADAR < 0 GOTO FOUND
   AIM TO RADAR
   IF RADAR < 0 GOTO AGAIN
   3 * I TO J
   AIM + J TO RADAR
   IF RADAR < 0 GOTO FOUND2
   0 - I TO I
   0 - J TO J
   AIM + J TO RADAR
   IF RADAR < 0 GOTO FOUND2
   ENDSUB
