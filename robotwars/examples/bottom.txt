

;     ROBOT  'BOTTOM'


; MOVE TO THE BOTTOM OF THE BATTLEFIELD

BOTTOM
  250 TO SPEEDY
  IF Y < 240 GOTO BOTTOM
  0 TO SPEEDY


; MOVE TO THE LEFT

MOVELEFT
  -100 TO SPEEDX
  GOSUB SCAN
  IF X > 20 GOTO MOVELEFT


; MOVE TO THE RIGHT

MOVERIGHT
  100 TO SPEEDX
  GOSUB SCAN
  IF X < 235 GOTO MOVERIGHT
  GOTO MOVELEFT


; SEARCH AND SHOOT

SCAN
  0 TO RADAR
  IF RADAR < 0, 0 - RADAR TO SHOT
  ENDSUB
